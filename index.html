Conceitos Básicos de HTML

HTML (Linguagem de Marcação de Hipertexto) é a base para criar páginas web.

Ele utiliza tags para estruturar o conteúdo, como textos, imagens e links.

As tags são delimitadas por < > e geralmente vêm em pares, como <p> e </p>.

Algumas tags são auto-fechamento, mas a maioria possui conteúdo entre abertura e fechamento.

Você pode destacar partes do texto usando a tag mark para chamar atenção.

Estrutura de um Documento HTML

Um documento HTML básico tem a seguinte estrutura:

  1. <!DOCTYPE html>: declara que o documento é HTML5.
  2. <html>: elemento raiz da página.
  3. <head>: contém metadados e título.
  4. <body>: onde vai o conteúdo visível.

Dentro do corpo (<body>), você pode usar várias tags para organizar o conteúdo.

Por exemplo, os cabeçalhos de <h1> a <h6> indicam níveis de títulos e subtítulos.

Também podemos usar listas ordenadas (

    ) e não ordenadas (
      ) para organizar itens.

Principais Tags HTML

  • <p>: parágrafo, para textos.
  • <a>: link, usado para navegar entre páginas ou seções.
  • <small>: texto pequeno, geralmente usado para informações secundárias.
  • <u>: texto sublinhado, útil para destacar.
  • <i>: texto em itálico, usado para ênfase leve.
  • <strong>: texto em negrito, para ênfase forte.
  • <mark>: destaca texto com fundo colorido.
  • <sub> e <sup>: texto subscrito e sobrescrito, para fórmulas e notas.
  • <blockquote>: citação longa de outras fontes.
  • <del>: texto deletado, que mostra alterações.
  • <font>: antiga tag para alterar cor e fonte (obsoleta, mas ainda presente).
  • <abbr>: abreviações com explicação ao passar o mouse.

Dica Rápida

Use sempre as tags semânticas para melhorar a organização do seu código e facilitar a vida de buscadores e leitores de tela.

Exemplo: use <header>, <nav>, <main>, <section>, <aside> e <footer>.
